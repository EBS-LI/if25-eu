# HINWEIS ZUM KRITISCHEN PFAD

Die folgenden drei Punkte koennen den gesamten Antrag gefaehrden, wenn sie
nicht rechtzeitig erledigt werden:

1. **Gesellschaftsgruendung + PIC-Registrierung + LEAR-Ernennung**
   Ohne PIC keine Einreichung. PIC-Registrierung wird vom Berater durchgefuehrt,
   aber der Geschaeftsfuehrer muss anschliessend die LEAR-Ernennung abschliessen.
   Beides zusammen dauert bis zu 2-3 Wochen.
   → Start: SOFORT

2. **Letter of Support Reverion mit TRL-Nachweis**
   Der Innovationsgrad ist ein Pass/Fail-Kriterium. Ohne belastbaren TRL-Nachweis
   faellt der Antrag bei der ersten Bewertungsstufe durch.
   → Reverion SOFORT kontaktieren

3. **Kostenvoranschlaege**
   Ohne belastbare Zahlen koennen weder das Financial Information File (FIF)
   noch die Machbarkeitsstudie fertiggestellt werden. Alles andere baut darauf auf.
   → Angebote bis spaetestens 20. Maerz einholen

# GLOSSAR / ABKUERZUNGSVERZEICHNIS

| Abkuerzung | Erklaerung |
|------------|------------|
| **BHKW** | Blockheizkraftwerk — Anlage zur gleichzeitigen Erzeugung von Strom und Waerme aus Biogas |
| **BImSchG** | Bundes-Immissionsschutzgesetz — Rechtsgrundlage fuer die Genehmigung von Biogasanlagen |
| **CAPEX** | Capital Expenditure — Investitionskosten (einmalige Anschaffungskosten) |
| **CINEA** | European Climate, Infrastructure and Environment Executive Agency — EU-Agentur, die den Innovation Fund verwaltet |
| **DNSH** | Do No Significant Harm — EU-Taxonomie-Prinzip: das Projekt darf keinem der 6 Umweltziele erheblich schaden |
| **EEG** | Erneuerbare-Energien-Gesetz — deutsches Gesetz zur Foerderung erneuerbarer Energien (inkl. Biogasverguetung) |
| **FIF** | Financial Information File — offizielle Excel-Vorlage der EU zur Darstellung der Finanzplanung (CAPEX, OPEX, Einnahmen, relevante Kosten) |
| **GHG** | Greenhouse Gas — Treibhausgas (CO2, CH4 etc.); im Kontext: GHG Emission Avoidance Calculator |
| **GPS** | Global Positioning System — Koordinatensystem zur Standortbestimmung (Breitengrad/Laengengrad) |
| **IF25 NZT** | Innovation Fund 2025 — Net-Zero Technologies; Foerderprogramm der EU fuer klimaneutrale Technologien |
| **IRR** | Internal Rate of Return — interner Zinsfuss; Renditekennzahl fuer Investitionsprojekte |
| **KfW** | Kreditanstalt fuer Wiederaufbau — deutsche Foerderbank (Programme 230, 270) |
| **LEAR** | Legal Entity Appointed Representative — von der EU validierte Kontaktperson der Organisation im Funding & Tenders Portal |
| **LoI** | Letter of Intent — Absichtserklaerung (hier: Zusage der Betreiber zu Biogaslieferung und Gesellschaftsbeitritt) |
| **LoS** | Letter of Support — Unterstuetzungsschreiben (hier: von Reverion GmbH als Technologiebestaetigung) |
| **Nm3/h** | Normkubikmeter pro Stunde — Masseinheit fuer Gasvolumenstrom unter Normbedingungen (0 °C, 1,013 bar) |
| **NPV** | Net Present Value — Kapitalwert; Barwert aller zukuenftigen Zahlungsstroeme eines Projekts |
| **OPEX** | Operational Expenditure — laufende Betriebskosten (Wartung, Personal, Energie, Versicherung) |
| **Part A** | Administrative Forms — administrativer Teil des EU-Antrags (Online-Formular im Portal) |
| **Part B** | Technical Description — technischer Hauptantrag (max. 70 Seiten, PDF-Upload) |
| **PG** | Projektgesellschaft — das neu zu gruendende gemeinsame Unternehmen der 8 Betreiber |
| **PIC** | Participant Identification Code — 9-stellige Kennnummer fuer Organisationen im EU Funding & Tenders Portal |
| **rSOC** | Reversible Solid Oxide Cell — Reverion-Technologie zur hocheffizienten CO2-Abscheidung aus Biogas |
| **RTO** | Regenerative Thermal Oxidizer — konventionelle Abgasreinigung (Referenztechnologie im Vergleich zu Reverion) |
| **TRL** | Technology Readiness Level — Technologiereifegrad (Skala 1-9, ab TRL 7 gilt eine Technologie als demonstriert) |
| **USt-ID** | Umsatzsteuer-Identifikationsnummer — EU-weit gueltige Steuernummer fuer Unternehmen |
| **UVP** | Umweltvertraeglichkeitspruefung — gesetzlich vorgeschriebene Pruefung der Umweltauswirkungen |
